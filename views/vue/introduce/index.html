# 介绍

# 准备

你需要在本地安装 node (opens new window) 和 git (opens new window)。 本项目技术栈基于 ES2015 (opens new window)、vue3 (opens new window)、vuex (opens new window)、vue-router (opens new window)、vue-cli (opens new window)、axios (opens new window)、element-plus (opens new window)和 sass (opens new window)，所有的请求数据都是真实接口响应，提前了解和学习这些知识会对使用本项目有很大的帮助。

本项目不支持低版本浏览器(如 ie)

# 运行

# 克隆项目
git clone https://github.com/gmingchen/vue3-element-plus-admin.git

# 进入项目目录
cd vue3-element-plus-admin

# 安装依赖
npm install

# 本地开发 启动项目
npm run dev

注意

建议不要用直接使用 cnpm 安装，可以通过配置 registry 来解决 npm 安装速度慢或中断的问题。

启动完成打开浏览器访问 http://localhost:9527，你看到下面的页面就代表操作成功了。
