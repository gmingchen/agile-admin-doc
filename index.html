# 介绍

vue3-element-plus-admin (opens new window) 是一个后台前端、后端解决方案，它基于 vue3 (opens new window) 和 element-plus (opens new window) 实现。使用了最新的前端技术栈，内置了 i18n 国际化，Typescript，动态路由，菜单权限管理，提炼了典型的业务功能，它可以帮助你快速搭建后台产品原型。

提示

master 分支默认使用 composition-api ，若需要 class 风格 + Typescript 请使用class-style (opens new window)分支，若需要 composition-api + Typescript + I18n 请使用composition-api (opens new window)分支，若只需要 基础版本 请使用base (opens new window)分支，但它们不会和 master 保持同步更新。

  • master (opens new window) 轻量版本：持续维护分支
  • base (opens new window) 精简版本：RBAC 权限功能-仅包含菜单管理、角色管理、用户管理三个模块
  • composition-api (opens new window) 内置Type Script、I18n
  • class-style (opens new window) 使用 class 风格开发，内置Type Script、I18n

说明

本文档只说明 master 分支，其他功能（如：I18n、TypeScript...）的使用请参考对应分支。
